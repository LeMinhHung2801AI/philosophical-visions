Chạy 3 Terminal khác nhau:
1. docker run -p 8000:8000 -v ./chroma-data:/data chromadb/chroma (chỉ một lần mỗi khi khởi động app)
(+) Tạo vector ban đầu: npx ts-node --project tsconfig.cjs.json scripts/ingest.cts (chỉ chạy lần đầu tiên, hoặc khi bạn thêm tài liệu mới) (philosophicalVisions\scripts\ingest.cts)

2. Terminal Backend: npx ts-node --project tsconfig.cjs.json server.cts (philosophicalVisions\server.cts)
3. Terminal Frontend: npm run dev

(+) Fix lỗi docker:
docker ps (Danh sách docker đang chạy)
docker stop [container_id] (Dừng tác vụ đang chạy)

(+) Fix lỗi thư viện (dependencies):
Xóa thư mục node_modules và file package-lock.json:
```
rd -Recurse -Force node_modules
del package-lock.json
```

npm install (Cài đặt lần đầu)
npm ci (Cài đặt chính xác phiên bản khi có sẵn file package-lock.json)
